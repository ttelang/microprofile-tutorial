Product Catalog Service

A microservice for managing product information in the e-commerce platform

Service Overview

The Product Catalog Service provides a REST API for managing product information, including:

  • Creating new products
  • Retrieving product details
  • Updating existing products
  • Deleting products
  • Searching for products by various criteria

MicroProfile Config Implementation

This service demonstrates MicroProfile Config capabilities, showcasing externalized configuration management and dependency injection of configuration properties.

Key MicroProfile Config Features

  • Property Injection - Configuration values injected directly into CDI beans using @ConfigProperty
  • Automatic Type Conversion - String values from config files automatically converted to Java types (boolean, int, Double)
  • Default Values - Fallback values provided when properties are not configured
  • Feature Flags - Boolean properties to enable/disable features at runtime
  • Hot Reload - Configuration changes detected automatically in liberty:dev mode

Configuration Properties

Property Name Type Default Purpose
product.maintenanceMode boolean false When true, service returns 503 responses for all API calls
product.catalog.name String E-Commerce Catalog Display name for the product catalog
product.resultsPerPage int 10 Number of products per page (demonstrates String to int conversion)
product.enableNewArrivals boolean true Feature toggle for new arrivals section
product.discount.percentage Double 10.0 Default discount percentage (visible in server logs)
mp.openapi.scan.disabled boolean false Enables automatic OpenAPI documentation scanning

Configuration Sources

MicroProfile Config supports multiple configuration sources with priority ordering:

  1. System properties (highest priority)
  2. Environment variables
  3. microprofile-config.properties file
  4. Default values specified in @ConfigProperty annotations (lowest priority)

Example: Maintenance Mode

The product.maintenanceMode property demonstrates a feature toggle pattern. When enabled, all API endpoints return HTTP 503 (Service Unavailable) responses.

To test this feature:

  1. Edit src/main/resources/META-INF/microprofile-config.properties
  2. Set product.maintenanceMode=true
  3. The application automatically reloads in liberty:dev mode
  4. Try accessing /api/products - you'll receive a 503 response

API Endpoints

Operation Method URL Description
List All Products GET /api/products Retrieves a list of all products
Get Product by ID GET /api/products/{id} Returns a product by its ID
Create Product POST /api/products Creates a new product
Update Product PUT /api/products/{id} Updates an existing product by its ID
Delete Product DELETE /api/products/{id} Deletes a product by its ID
Search Products GET /api/products/search Search products by criteria (name, description, price range)

Sample Usage

List All Products

GET /api/products

Response:

[
  {
    "id": 1,
    "name": "Smartphone X",
    "description": "Latest smartphone with advanced features",
    "price": 799.99
  },
  {
    "id": 2,
    "name": "Laptop Pro",
    "description": "High-performance laptop for professionals",
    "price": 1299.99
  }
]

Get Product by ID

GET /api/products/1

Response:

{
  "id": 1,
  "name": "Smartphone X",
  "description": "Latest smartphone with advanced features",
  "price": 799.99
}

Create a New Product

POST /api/products
Content-Type: application/json

{
  "name": "Wireless Earbuds",
  "description": "Premium wireless earbuds with noise cancellation",
  "price": 149.99
}

Response:

{
  "id": 3,
  "name": "Wireless Earbuds",
  "description": "Premium wireless earbuds with noise cancellation",
  "price": 149.99
}

Update a Product

PUT /api/products/3
Content-Type: application/json

{
  "name": "Wireless Earbuds Pro",
  "description": "Premium wireless earbuds with advanced noise cancellation",
  "price": 179.99
}

Response:

{
  "id": 3,
  "name": "Wireless Earbuds Pro",
  "description": "Premium wireless earbuds with advanced noise cancellation",
  "price": 179.99
}

Delete a Product

DELETE /api/products/3

Response: No content (204)

Search for Products

GET /api/products/search?name=laptop&minPrice=1000&maxPrice=2000

Response:

[
  {
    "id": 2,
    "name": "Laptop Pro",
    "description": "High-performance laptop for professionals",
    "price": 1299.99
  }
]
